# CUPE Local 2 TTC Campaign Website

A one-page campaign website built with HTML, CSS, and vanilla JavaScript to support CUPE Local 2's call for fair TTC service.

## Files

- `index.html` — page structure and campaign content
- `styles.css` — styling, accessibility states, and responsive layout
- `script.js` — email form validation/success message + dynamic footer year

## Run locally

1. Clone or download this repository.
2. Open `index.html` in your browser.

That's it — no build tools or dependencies are required.
